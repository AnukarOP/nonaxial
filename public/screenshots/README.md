# Screenshots for README

Add your screenshots here. Use these tools to create macOS-style mockups:

## Tools
- **[Screely](https://screely.com)** — Instant macOS window frame
- **[BrowserFrame](https://browserframe.com)** — Browser chrome overlay
- **[CleanShot X](https://cleanshot.com)** — macOS app with window capture
- **[LICEcap](https://www.cockos.com/licecap/)** — Record GIFs on Windows/Mac

## Tips
1. Use dark mode for consistent look
2. Record GIFs at 15-30fps to keep file size small
3. Add subtle shadow in Screely for depth
4. Keep GIFs under 2MB each
